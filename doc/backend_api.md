# 获取仓库信息表API
**请求方法**: GET  
**请求路径**: /api/users  
**请求参数**:
```json
{
  "table": "goods"
}
```

**返回示例**:
```json
{
  "日志ID",
  "日志ID",
  "日志ID",
  "日志ID",
  "日志ID",
  "日志ID",
}
```
